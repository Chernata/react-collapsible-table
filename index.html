Our react table is expandable filterable sortable awesome!

This is our awesome react table. Check out the example, grab the code, and help us make it even better!

Features

Expandable rows

We wanted to prioritise the information users see in our table, so our rows expand to show more underlying data.

Filterable columns

We wanted to let our users remove any information they might not need to see, so columns can be filtered on or off.

Design built in

Our table is designed out of the box to follow table best practices. Lining & tabular numbers, correct alignment, and horizontal rules are built in.
